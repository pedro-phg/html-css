Escrever códigos

Código não formatado

Para inserir textos em formato de código, isto é, um texto em que cada caracter ocupa o mesmo espaçamento, utilizamos a tag 'code'.

print("Olá, Mundo!")

Código Formatado

Para escrever um código formatado, devemos utilizar a tag 'code' dentro da tag 'pre'

        
    num = int(input("Digite um número: "))
    if num % 2 == 0

Citações Simples

As citações simples servem para colocar um texto entre aspas duplas. Porém, semânticamente, significam mais do que simplemesmente colocar o texto entre aspas enquanto escrevemos um código. Para isso, utilizamos o comando 'q'

Como diria o pai de um amigo: O Computador é um burro muito rápido.

Citações Completas

As citações completas servem para que possamos referenciar (dentro de nosso código) algum texto disponível na internet. As buscas por esse texto possivelmente vão levar ao seu site também. Para isso, utilizamos o comando 'blockquote'.

Abreviações

Para criar uma sigla que contenha uma abreviação, basta utilizarmos o comando abbr. Dentro deles colocamos o significado da sigla.

IBGE ENEM MEC

Listas

Existem dois tipos de listas, as Listas Ordenadas (ordered lists) e as Listas não Ordenadas (un-ordered lists).

Para criar listas ordenadas utilizamos a tag ol. Para criar listas não ordenadas utilizamos a tag ul.

Listas Ordenadas

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

Listas Sem Ordem

  • item 1
  • item 2
  • item 3
  • item 4
